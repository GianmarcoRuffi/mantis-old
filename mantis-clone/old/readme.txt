MINIMUM SYSTEM REQUIREMENTS
___________________________________

Pentium Class PC - 200MHz or faster 
 - Windows® 98 Second Edition, Me, 2000 or XP
 - Internet Explorer 5.01 or later
 - 64 MB RAM, 96 MB recommended
 - 115 MB available hard disk space, 
    40 MB additional hard disk space during setup (155 MB total)
 - 256 colors
 - 800x600 screen resolution
 - CD-ROM drive
 - Mouse or compatible pointing device
 
Internet access required for Tournament Reporting to Upper Deck
Network access required for networking functionality



INSTALLATION
___________________________________

The Mantis installation program will (a) update your system if required, (b) install the Mantis program and (c) instruct you to install the .NET Framework 1.1 if required.  Previous installations (including beta versions) of the Mantis program will need to be deleted through "Add/Remove Programs" which will not delete your database file, "Mantis.mdb".

1. Insert the CD into your CD ROM drive.
2. If the installer does not automatically launch, 
   select the "Start" button, then "Run", 
   Browse to the CD and select "Setup.exe".
3. Follow the on-screen instructions to complete the 
   installation. If your computer reboots during setup, 
   leave the CD in the CD-ROM drive.  If the setup 
   program does not automatically continue, please 
   repeat Step 2 above.



RUNNING THE PROGRAM
___________________________________

Click the "Start" Button, select "Programs" or "All Programs", select "Upper Deck", then select the "Mantis" folder icon, and select the "Mantis" program icon.  You may also double-click the "Mantis" icon installed on your desktop.



TROUBLESHOOTING
___________________________________

Error:
Unable to install Mantis.

Solution:
A User must have Administrator Privileges in order to properly install Mantis.


Error:
During the .NET setup, this message appears:
"To continue with setup it is necessary to reset the Windows Installer. Restart your computer and start setup again."

Solution:
You will need to complete the following steps:
1. Restart your computer.
2. Remove & re-insert the Mantis CD to restart the Mantis setup program.
3. Select Repair.
4. Select yes to install .NET Framework 1.1


Error:
Mantis won't install in a language other than English.
 
Solution:
If a language selection window appears select English for the installation then select the language of your choice while running Mantis (Utilities - Set Language).
If a language selection window does not appear - Browse to the CD and select "Mantis_Install.msi" to install in English then select the language of your choice while running Mantis (Utilities - Set Language).


Error:
When attempting to start Mantis, one of the following messages appear:
"A required .DLL file, MSCOREE.DLL, was not found"
"The application failed to initialize properly (0xc0000135). Click on OK to terminate the application."

Solution:
Your computer needs the Microsoft .NET Framework 1.1 to be installed.  Install it from the Mantis CD-ROM. Click the "Start" Button, select "Run..." then type:
X:\utilities\dotnetfx.exe
where "X:" is the letter of your CD-ROM drive.  If you do not have the CD-ROM, download it from Microsoft's web site at:
http://www.microsoft.com/downloads/details.aspx?familyid=262d25e3-f589-4842-8157-034d1e7cf3a3&displaylang=en


Error:
While running Mantis, this message appears:
"An unhandled exception has occurred in your application.  If you click Continue, the application will ignore this error and attempt to continue.  If you click Quit, the application will be shut down immediately.
The .Net Data OLE DB Provider(System.Data.OleDb) requires Microsoft Data Access Components(MDAC) version 2.6 or later."

Solution:
Your computer needs version 2.6 or later of the Microsoft Data Access Components installed. Reinstall or Repair the Mantis Software and it will update your MDAC software.  You may also install it from the Mantis CD-ROM.  Click the "Start" Button, select "Run..." then type:
X:\utilities\MDAC_TYP.EXE
where "X:" is the letter of your CD-ROM drive. If you do not have the CD-ROM, download it from:
http://www.microsoft.com/downloads/details.aspx?FamilyID=6c050fe3-c795-4b7d-b037-185d0506396c&DisplayLang=en


Error:
Documents labeled "(pdf)" do not open correctly.

Solution:
Your computer needs Adobe Acrobat Reader installed.  You may install the software from the Mantis CD-ROM.  Click the "Start" Button, select "Run..." then type:
X:\utilities\AcrobatReader60_enu_full.exe
where "X:" is the letter of your CD-ROM drive. If you do not have the CD-ROM, download it from:
http://www.adobe.com/products/acrobat/readstep2.html


Error:
When attempting any uplink activity, the login fails and a popup message appears: 
"Failed to connect. If you are behind a firewall try setting the proxy settings."

Solution:
If you are behind a firewall, close the uplink dialog box and select UTILITIES then OPTIONS from the pull down menu. In the Options dialog box click proxy settings and then enter the proxy settings. 
Or ... If your ISP has provided an accelerator service, disable or uninstall this software and try uplink again. Contact your ISP to determine the software and how to disable it.
Or ... If you do not have Tournament Organizer status from UDE you will not be able to uplink and send or receive data. Contact UDE if you believe this to be the cause.
Or ... Wait and try again. It could be that the service to connect to UDE is down for maintenance.


Error:
During installation, the following message appears:
"Setup has created an  error in DIRAPI.DLL. The Setup program will be aborted. If you still encounter problems, try to restart your computer"

Solution:
You need to manually launch the Mantis installation program. Click the "Start" Button, select "Run..." then type:
X:\utilities\Mantis_Install.exe
where "X:" is the letter of your CD-ROM drive.


Error:
Archive tournament list information is incomplete

Solution:
Tournament status (reported or complete) and the date a tournament is reported; will be listed in the Archive screen when Mantis is restarted.
